📘 如何部署 Cloudflare Worker？（點擊展開）
  1. 開啟專案內的 backend/ 資料夾 → 閱讀 README.md
  2. 安裝 Wrangler：cd backend && npm install
  3. 登入 Cloudflare：npx wrangler login
  4. 建立 KV Namespace：npx wrangler kv namespace create MEMBERS → 把回傳的 id 填入 wrangler.toml
  5. 設定 Secret（三個）：
    • npx wrangler secret put LINE_CHANNEL_ACCESS_TOKEN
    • npx wrangler secret put LINE_CHANNEL_SECRET
    • npx wrangler secret put ADMIN_TOKEN（自己設一組密碼）
  6. 部署：npx wrangler deploy → 拿到 Worker URL
  7. 回到這裡把 Worker URL + ADMIN_TOKEN 填進去
  8. 最後到 LINE Developers 的 Messaging API Channel 設定 Webhook URL：https://你的.workers.dev/api/webhook

📱 LINE 官方帳號・整合設定 尚未設定

平台支援綁定 LINE 官方帳號。填入以下資訊後，使用者透過 LINE Rich Menu 開啟時將自動登入並取得個人資料。
取得方式：LINE Developers Console → 建立 LIFF App → 複製 LIFF ID
格式：10 位數字 + 連字號 + 8 位英數
📘 如何取得 LIFF ID？（點擊展開）
  1. 到 LINE Developers Console 登入
  2. 建立 Provider（若已有可跳過）
  3. 在 Provider 下 Create Channel → 選 LINE Login
  4. 進入 Channel → 點上方 LIFF 分頁 → Add
  5. 填入：
    　• Size：Full
    　• Endpoint URL：https://onelifegpt-tech.github.io/sunmoonlake-lowcarbon/
    　• Scopes：勾 profile + openid
    　• Scan QR：On
  6. 建立後會看到藍色的 LIFF ID，格式如 2001234567-AbCdEfGh，複製貼到上面的欄位即可
