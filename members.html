⚙️ 正式推播需要的設定

1. LINE Developers Console 建 Messaging API Channel（跟 LIFF 的 LINE Login Channel 不一樣）
2. 將 Channel 綁到 LINE 官方帳號
3. 取得 Channel Access Token
4. 後端伺服器用 POST 呼叫 https://api.line.me/v2/bot/message/broadcast
5. 在 .env 填入 Token 即可推播
